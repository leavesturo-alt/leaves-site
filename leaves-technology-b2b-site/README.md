# LEAVES Technology B2B Website

This is a Next.js + Tailwind website focused on B2B fleet operations for Turo hosts.

## Run locally
1. Install Node.js 18+
2. In this folder run
   npm install
   npm run dev
3. Open http://localhost:3000

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Import into Vercel
3. Set your domain
4. In app/layout.tsx update metadataBase from https://example.com to your domain
